# kadabite-KadabiteAPI


# Building project
```bash
    pnpm run build
    pnpm run start
```

# To Restart the app
```bash
    pm2 restart kadabites-api
```

# To delete the running app from pm2
```bash
    pm2 delete kadabites-api
```

# To stop the running app
```bash
    pm2 stop kadabites-api
```
